预备知识HTML表格

在 HTML 中一个很普通的任务是构建表格数据，有大量的元素和属性是来满足这种需求的。只需要一点儿的 CSS 来设定风格，HTML 让在 web 上显示表格数据变的很容易，例如你的学校的教学计划，你当地的游泳馆的时刻表，或者是关于你最爱的恐龙或足球队的统计数据。这个模块会教给你所有你需要知道的关于用 HTML 构建表格数据的知识。
